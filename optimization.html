One-Dimensional Optimization

Fix x0∈Rx_0 \in \Rx0​∈R and let f:[x0,∞)→Rf: [x_0, \infty) \to \Rf:[x0​,∞)→R be a continuous function. We would like to devise an algorithm to find a minimizer x∗x^*x∗ of fff that is, in some sense, “near” x0x_0x0​.

min⁡x≥x0f(x) \min_{x \ge x_0} f(x) x≥x0​min​f(x)

We will not assume any particular form for the function fff, and we will develop algorithms that only access fff as a black box. In other words, given a real number x∈[x0,∞)x \in [x_0, \infty)x∈[x0​,∞), we can call a subroutine that returns the numerical value of f(x)f(x)f(x). This output value is the only feature of the subroutine that we can observe; we cannot otherwise look inside to see what it is doing.

Under these conditions, any algorithm that terminates can only perform a finite number of evaluations of fff. Hence, no algorithm of this type can provide any guarantees about finding a global minimizer or computing an exact result. Even in the one-dimensional case, the best we can hope to do is to approximate a local minimizer of fff.
