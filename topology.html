Definition: Pseudometric, Symmetry, Triangle Inequality

Let XXX be a set, and let FFF be an ordered field. An FFF-valued pseudometric on XXX is a function d:X×X→Fd: X \times X \to Fd:X×X→F that satisfies the following requirements:

  • d(x,x)=0Fd(x, x) = 0_Fd(x,x)=0F​ for all x∈Xx \in Xx∈X.
  • Symmetry: d(x,y)=d(y,x)d(x, y) = d(y, x)d(x,y)=d(y,x) for all x,y∈Xx, y \in Xx,y∈X.
  • Triangle inequality: d(x,z)≤d(x,y)+d(y,z)d(x, z) \le d(x, y) + d(y, z)d(x,z)≤d(x,y)+d(y,z) for all x,y,z∈Xx, y, z \in Xx,y,z∈X.

When the ordered field FFF is not explicitly specified, it is usually assumed to be the ordered field R\RR of real numbers.

Pseudometrics are Nonnegative

Theorem: Let XXX be a set, let FFF be an ordered field, and let d:X×X→Fd: X \times X \to Fd:X×X→F be an FFF-valued pseudometric on XXX. For all x,y∈Xx, y \in Xx,y∈X, we have d(x,y)≥0Fd(x, y) \ge 0_Fd(x,y)≥0F​.

Proof: Let x,y∈Xx, y \in Xx,y∈X be given. By the defining properties of a pseudometric, we have

0F=d(x,x)≤d(x,y)+d(y,x)=2Fd(x,y) 0_F = d(x, x) \le d(x, y) + d(y, x) = 2_F d(x, y) 0F​=d(x,x)≤d(x,y)+d(y,x)=2F​d(x,y)

which implies that d(x,y)≥0Fd(x, y) \ge 0_Fd(x,y)≥0F​. (Note that 2F≠0F2_F \ne 0_F2F​=0F​ in all ordered fields FFF.) ∎

Definition: Metric, Distance Function, Positivity

Let XXX be a set, and let FFF be an ordered field. An FFF-valued metric on XXX, also known as an FFF-valued distance function on XXX, is an FFF-valued pseudometric d:X×X→Fd: X \times X \to Fd:X×X→F that satisfies the following additional requirement:

  • Positivity: For all x,y∈Xx, y \in Xx,y∈X, if d(x,y)=0Fd(x, y) = 0_Fd(x,y)=0F​, then x=yx = yx=y.

In other words, the only difference between a pseudometric and a metric is that a pseudometric is allowed to assign an distance of zero to two distinct points, while a metric must assign positive distances between distinct points.

Definition: Generalized Pseudometric Space, Underlying Set, Distance Field, Point

A generalized pseudometric space is an ordered triple (X,F,d)(X, F, d)(X,F,d) consisting of a set XXX, called the underlying set, an ordered field FFF, called the distance field, and an FFF-valued pseudometric ddd on XXX. The elements of XXX are called points.

Definition: Topology

Let XXX be a set. A topology on XXX is a collection T⊆P(X)T \subseteq \powerset{X}T⊆P(X) of subsets of XXX that satisfies the following requirements:

  • Empty set is open: ∅∈T\varnothing \in T∅∈T.
  • Whole set is open: X∈TX \in TX∈T.
  • Closed under arbitrary unions: If S⊆TS \subseteq TS⊆T, then ⋃S∈T\bigcup S \in T⋃S∈T.
  • Closed under finite intersections: If S⊆TS \subseteq TS⊆T and ∣S∣\abs{S}∣S∣ is finite, then ⋂S∈T\bigcap S \in T⋂S∈T.

Definition: Topological Space, Point, Open Set

A topological space is an ordered pair (X,T)(X, T)(X,T) consisting of a set XXX, called the underlying set, and a topology TTT on XXX. The elements of XXX are called the points of the topological space (X,T)(X, T)(X,T), and the elements of TTT are called its open sets.

Definition: Closed Set

Let (X,T)(X, T)(X,T) be a topological space. A subset F⊆XF \subseteq XF⊆X is closed if X∖F∈TX \setminus F \in TX∖F∈T.

In other words, a set is closed if and only if its complement is open. Because taking complements is an involution (i.e., X∖(X∖A)=AX \setminus (X \setminus A) = AX∖(X∖A)=A for all A⊆XA \subseteq XA⊆X), the converse is also true: a set is open if and only if its complement is closed.

Note that, in topology, “open” and “closed” are not opposites! It is possible for a set to be both open and closed, and it is also possible for a set to be neither. In fact, there is a special name for sets that are both open and closed.

Definition: Clopen Set

Let (X,T)(X, T)(X,T) be a topological space. A subset of XXX is clopen if it is both open and closed.

Every topological space (X,T)(X, T)(X,T) has at least two clopen sets: the empty set ∅\varnothing∅ and the set XXX itself.
